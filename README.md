# fx_graph

### 1. fx_data_csv.py  為替データを取得し、CSVへ変換する
### 2. fx_select       CSVから指定した期間のデータを抽出する
### 3. fx_graph        抽出したデータから Backtesting.py を用いてバックテストを行なう

　

> バックテストとは
> 
> システムトレード（売買ルール）の有効性を検証する際に、過去のデータを用いて、  
> 一定期間にどの程度のパフォーマンスが得られたかをシミュレーションすることです。
